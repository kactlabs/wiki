# How to install FFMPEG

**Note:** notes



FFMPEG in Ubuntu 18.04
```
sudo apt update
sudo apt install ffmpeg
ffmpeg -version
```

Print all encoders
```
ffmpeg -encoders
ffmpeg -decoders
```
