# How to install Min Browser

**Note:** Go Minimum




```
How to install Min browser?

wget https://github.com/minbrowser/min/releases/download/v1.20.0/min_1.20.0_arm64.deb

sudo dpkg -i min_1.20.0_arm64.deb

sudo apt-get update
sudo apt-get install gconf2

sudo apt --fix-broken install
```
